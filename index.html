Quick Demo

Sibling: +

			div+p+bq
		
			<div></div>
			<p></p>
			<blockquote></blockquote>

Multiplication: *

			ul>li*3
		
			<ul>
				<li></li>
				<li></li>
				<li></li>
			</ul>

Grouping: ()

			(ul>li*2>a)+footer
		
			<ul>
				<li><a href=""></a></li>
				<li><a href=""></a></li>
			</ul>
			<footer>
			</footer>

ID and CLASS

			div#header+div.page+div#footer.class1.class2.class3
		
			<div id="header"></div>
			<div class="page"></div>
			<div id="footer" class="class1 class2 class3"></div>

Custom attributes

			td[title="Hello world!" colspan=3]
		
			<td title="Hello world!" colspan="3"></td>

Item numbering: $

			ul>li.item$*5
		
			<ul>
				<li class="item1"></li>
				<li class="item2"></li>
				<li class="item3"></li>
				<li class="item4"></li>
				<li class="item5"></li>
			</ul>

Text: {}

			a{Click me}
		
			<a href="">Click me</a>
